# movieRapp

A simple Movie Review Application built using React JS.  
Users can browse movies, search and filter them, view detailed information, and rate movies using a star-based rating system.
This project focuses on data display, input handling, and basic user interaction

#Features
-Browse and search for movies easily  
- Filter movies by genre or year  
- View detailed information of selected movies  
- Rate movies using a star-based rating system  
- Responsive and clean UI using Tailwind CSS
